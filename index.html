How this estimator works

Static-host friendly

This site is designed for GitHub Pages, so this version scans supported live sources with browser-safe fetches instead of a private backend that indexes the whole web.

Import estimate, not customs clearance advice

The India landed-cost model is an estimate. Real customs duty can change by category, courier handling, exemptions, restrictions, and customs classification.
